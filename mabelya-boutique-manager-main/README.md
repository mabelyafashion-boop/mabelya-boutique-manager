# 🛍️ MABELYA FASHION – Boutique Manager

## 📌 Présentation

Application web de gestion de boutique permettant de suivre efficacement les ventes, les produits, les clients, les dépenses et le personnel.

---

## 🌐 Accès à l’application

👉 https://mabelya-boutique-manager-36ln347jt.vercel.app/

---

## ⚙️ Fonctionnalités

* Gestion des produits
* Gestion des ventes
* Gestion des clients
* Suivi des dépenses
* Gestion du personnel
* Tableau de bord

---

## 🧱 Technologies utilisées

* React
* TypeScript
* Tailwind CSS
* Backend : Supabase
* Déploiement : Vercel

---

## 💻 Installation en local

### Prérequis

* Node.js

### Lancer le projet

```bash
npm install
npm run dev
```

---

## 🔐 Configuration

Créer un fichier `.env.local` :

```env
NEXT_PUBLIC_SUPABASE_URL=xxxxx
NEXT_PUBLIC_SUPABASE_ANON_KEY=xxxxx
```
